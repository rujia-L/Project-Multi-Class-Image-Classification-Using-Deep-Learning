# Project-Multi-Class-Image-Classification-Using-Deep-Learning
Comparative deep learning study for 1,000-class Pokémon image classification (26,539 images) using Custom CNN, MobileNetV2, and ResNet50; achieved 87.37% test accuracy with ResNet50.

# Data source:
https://www.kaggle.com/datasets/noodulz/pokemon-dataset-1000/data
